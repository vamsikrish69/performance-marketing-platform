
# 🚀 Performance Marketing Data Platform (dbt + BigQuery)

## 📌 Overview
This project builds an end-to-end performance marketing data platform using dbt and BigQuery to analyze campaign performance.

---

## 🧱 Architecture

- Staging → Data cleaning
- Intermediate → Data transformation & joins
- Marts → Final KPI tables

---

## 📊 Key KPIs

- ROAS (Return on Ad Spend)
- CAC (Customer Acquisition Cost)
- CTR (Click-Through Rate)
- ROI (Return on Investment)

---

## 📂 Final Tables

- pf_fct_campaign_performance
- pf_fct_customer_acquisition
- pf_fct_channel_performance

---

## 🛠 Tech Stack

- dbt
- BigQuery
- SQL

---

## 💡 Description

This platform integrates Meta Ads, Google Ads, conversions, and customer data to generate business-ready insights for marketing performance analysis.
